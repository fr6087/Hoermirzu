sets the enabled property of a TextBox control to disabled, if the model Element's property is set to 'disabled' string. This is the case for input elements that are only for displaying information, e.g. summing up other entries made by the user. If the TextBox stays enabled, the header will be added to the HashTable that is used by the LUIS-Model to determine to which TextBox to pass a user value to.

Parameters

elementthe model insatnce that saved the future property valus of the field
