async Task ListenToMe.Model.Form.Fill_sections ( )

calls the wcf service to dertermine the structure of the form. Then, it separates the structure into sections and binds them to a static attibute that's globally available in ListenToMe App. To which object it outputs depends on the debug mode. If debug is enabled, it will write to sectionsList. If not it will write to a Dictionary.
