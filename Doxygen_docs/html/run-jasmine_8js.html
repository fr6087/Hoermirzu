function waitFor (   testFx,
  onReady,
  timeOutMillis 
)

Wait until the test condition is true or a timeout occurs. Useful for waiting on a server response or for a ui change (fadeIn, etc.) to occur.

Parameters
testFxjavascript condition that evaluates to a boolean, it can be passed in as a string (e.g.: "1 == 1" or "$('#bar').is(':visible')" or as a callback function.
onReadywhat to do when testFx condition is fulfilled, it can be passed in as a string (e.g.: "1 == 1" or "$('#bar').is(':visible')" or as a callback function.
timeOutMillisthe max amount of time to wait. If not specified, 3 sec is used.
