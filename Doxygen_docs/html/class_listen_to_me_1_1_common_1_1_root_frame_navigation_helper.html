Detailed Description

registers for standard mouse and keyboard shortcuts used to go back and forward. There should be only one instance of this class per view, and it should be associated with the root frame.

To make use of RootFrameNavigationHelper, create an instance of the RootNavigationHelper such as in the constructor of your root page.

public MyRootPage()
{
this.InitializeComponent();
this.rootNavigationHelper = new RootNavigationHelper(MyFrame);
}
